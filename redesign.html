Redesign Page:

Home Page

My home page was completely overhauled with the introduction of containers that now give the page a more professional feel. In addition I used the nav component in replacement of my CSS nav bar. I customized the font and overall color scheme as well.

About Me

My new about me page features a carousel component that cycles 3 pictures of myself. There are also buttons on the sides of the images so the user can quickly click through all 3.

Portfolio

My portfolio page now uses images for links as opposed to just text hyperlinks. I also used the columns component to display the project icons horizontally. I used the same theme and color as the other pages in my website.

Contact

The contact page I wanted to keep relatively simple. Instead of just listing the contact info I decided to play with the select feature and included a link to my LinkedIn page. When the user selects the linkedin option from the dropdown options, it will automatically redirect to my linked in page as opposed to having to select it, copy and paste it and then hit search in a browswer. I figure the less clicks the better!
