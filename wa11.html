What is this chart?

Glad you asked. This chart is an area chart. This type of chart uses the filled portion of theg raph to show changes in quantitative values of the data. Using the interaction tool you can decide how you want to interpolate the data(i.e : show the changes linearly, only cardinal values, on a step basis etc...)
